Lead · Model Release

It beats GPT-5.4 and Gemini 3.1 Pro on the benchmarks that matter most to you.

Opus 4.7 arrives, and it was built for orchestrators.

Source · Anthropic Shipped · April 16, 2026 Price · $5 / $25 per M tokens

Anthropic's most capable generally available model is now sitting in front of every Claude Code user by default, and it is unusually aggressive about it — Claude Code now runs Opus 4.7 at the new xhigh effort level out of the box on every plan. That single default change is the real headline. Opus 4.7 is no longer a model you opt into for hard problems; it is the floor.

The numbers explain the confidence. SWE-bench Verified jumps from 80.8% to 87.6%. SWE-bench Pro — the harder private variant — goes from 53.4% to 64.3%, beating GPT-5.4's 57.7%. CursorBench climbs from 58% to 70%. Anthropic reports a 14% improvement in multi-step agentic reasoning with roughly a third as many tool errors as the previous generation. The vision tower tripled in resolution to 3.75 megapixels, which finally makes computer-use agents reliable on dense UI screenshots.

Two feature additions deserve specific attention from anyone running production agent pipelines. Task budgets, in public beta on the Claude Platform API, let you cap the token spend on a single long-running task so parallelized agents don't silently blow through margin. And the new /ultrareview command in Claude Code runs a structured self-critique pass over a diff before the model claims it is done — a small thing that moves a lot of eng time off of reviewing Claude's homework.

Deal · Infrastructure

Amazon doubles down on Anthropic with another $25B — and a $100B AWS lock-in.

The cloud alignment is now fully drawn. Amazon announced on Monday it will invest up to $25B more in Anthropic — $5B immediately at the existing $380B valuation, plus up to $20B tied to commercial milestones — stacked on top of the $8B it had already put in. In return, Anthropic has committed to spend more than $100 billion on AWS over the next decade and to run its frontier models on AWS Trainium silicon for at least that long.

Read between the lines: this is Amazon's answer to Microsoft–OpenAI and Google's in-house stack, and it turns Anthropic into a ten-year demand engine for Trainium. Anthropic will bring almost one gigawatt of combined Trainium2 and Trainium3 capacity online before year end. The full Claude Platform will also be available natively inside AWS — existing billing, IAM, VPCs — rather than living solely behind the Bedrock marketplace.

A ten-year commitment on custom silicon is the strongest signal yet that Anthropic never intends to be a GPU company.

Critical · Supply Chain MCP Advisory

OX Security · The Register · Tom's Hardware

The MCP supply chain has a 200,000-server problem.

OX Security's April 15 disclosure is the first systemic security finding against MCP — and it is not a bug. It is a design decision. The official Anthropic MCP SDKs in Python, TypeScript, Java, and Rust accept arbitrary command strings on the STDIO transport and execute them regardless of whether the server process successfully starts. Feed the loader a malicious command, receive an error, and the command has already run.

The blast radius is the part that changes how you treat it. Ox counts 200+ open-source projects, 150M downloads, 7,000+ publicly reachable servers and up to 200,000 vulnerable instances — flagship tools like LiteLLM, LangFlow, Windsurf, Cursor, Flowise, DocsGPT and GPT Researcher are all affected. Four exploit families have been demonstrated, including zero-click prompt injection inside AI coding IDEs.

Anthropic's position, confirmed to The Register, is that STDIO's behavior is expected and that sanitization belongs to the developer. That answer is defensible from a protocol-designer's chair. It is also the moment MCP's governance question — now sitting under the Linux Foundation's Agentic AI Foundation — stops being theoretical.

"Expected behavior" is how supply-chain fires start.

Anthropic · AISI · Foreign Policy

Project Glasswing, explained
Safety · Cyber

Claude Mythos is the first model that changed the cyber calculus.

Anthropic previewed Mythos to about fifty partner organizations on April 7 — AWS, Apple, Google, Microsoft, JPMorgan, NVIDIA, the Linux Foundation, CrowdStrike, Palo Alto, and a handful of government bodies. In the weeks since, Mythos has reportedly found thousands of previously unknown high-severity vulnerabilities across every major operating system and every major browser. The UK AI Security Institute concluded Mythos is "strikingly capable" on offensive cyber tasks that, until now, separated top human red teams from everyone else.

This is the moment the defenders-first narrative collides with reality. Project Glasswing's premise is that if frontier AI can find zero-days at scale, the ethical move is to find them first and patch them into the software the world runs on before the model — or something like it — gets into adversarial hands. Anthropic has privately warned U.S. officials that Mythos makes large-scale autonomous cyberattacks materially more feasible. The AISI evaluation doesn't disagree.

Imminent · OpenAI

"Spud" is almost here — and Polymarket thinks it ships today.

OpenAI's next foundation model, internally codenamed Spud and widely expected to ship as GPT-5.5, finished pretraining on or around March 24. Sam Altman has confirmed it internally as "a very strong model" that could "really accelerate the economy," and references to it have already leaked out of the Codex tool and the ChatGPT interface. Polymarket traders as of Wednesday evening were pricing an 86% probability that Spud ships on April 23 — i.e., today — following Altman's "really excited for this week" tease and a scheduled noon-PT livestream.

What's been telegraphed so far: a reworked agentic runtime (native computer use built into the base model, not bolted on by an SDK), tighter tool-use primitives meant to undercut Claude Code on its home turf, and a unification story that finally merges the GPT-4o lineage with the reasoning line. If the benchmarks land where Altman implies, the pressure pops back to Anthropic within the week. Watch whether Opus 4.7's SWE-bench Pro lead survives the day.

Private Markets · AI Coding

Cursor is raising $2B at $50B — and SpaceX may buy the whole thing for $60B anyway.

Cursor, now formally Anysphere, entered the week in the kind of position that stops being about fundraising and starts being about auction dynamics. Bloomberg first reported on April 17 that the company was in talks to raise $2B at a valuation north of $50B, co-led by Andreessen Horowitz with Nvidia and Thrive Capital. By February, Cursor's annual run rate had already crossed $2B, making it the fastest B2B software company in history to hit $1B ARR — faster than Slack, Zoom, or Snowflake.

Then the room shifted. Bloomberg followed up Wednesday with a line that recontextualizes the entire round: Elon Musk's SpaceX is in talks to acquire Cursor outright for $60B. A16z and Thrive are staring at windfalls on the order of a 50× from their early checks. If the acquisition closes, it is the largest private software exit in history — and it locks Cursor's AI coding stack into Musk's orbit alongside xAI. For the dozen teams still deciding between Cursor and Claude Code, the deal matters less for the multiple and more for the strategic question: whose roadmap do you want your editor on?

Fastest to $1B ARR in the history of B2B software. Now being bid on by the world's most controversial buyer.

Anthropic · SiliconANGLE · InfoWorld

Product · Agents

Claude Managed Agents makes Anthropic the hosting layer, not just the model.

The April 8 public-beta launch of Claude Managed Agents is the quietest move with the loudest implication in this issue. For the first time, you can define an agent's task, tools, and guardrails, then hand it off to Anthropic to run — sandboxed containers, built-in tools, streaming events — with Anthropic handling the second job entirely. Pricing starts at $0.08 per runtime hour on top of standard token costs, which is about $58 a month for a 24/7 agent before inference.

Combined with the public Agent Skills specification — the same packaged-skill format you are already using in Cowork — Managed Agents closes the loop: specialize with a Skill, deploy as a Managed Agent, observe through the Claude Platform. It is not an accident that this shipped two weeks before Opus 4.7 turned xhigh on by default. The ambition is the full agent stack on Anthropic's rails, with AWS underneath.

The model is the easy part. Running it reliably for months on end is what Anthropic just took off your plate.

Alibaba and Tencent are reportedly first in.

Funding · China

DeepSeek opens its first-ever fundraise at a $20B valuation.

DeepSeek, which spent two years operating as the world's most famous self-funded frontier lab, has finally broken and is raising outside capital. Bloomberg and The Information report the round is targeting a valuation north of $20B, with both Alibaba and Tencent circling. For context: that is about one-nineteenth of Anthropic's current mark and roughly one-fortieth of OpenAI's last print, despite DeepSeek's models landing inside 2–3 points of the frontier on most public evals.

The twist is political. A Chinese lab accepting strategic capital from China's two largest internet platforms — right as the U.S. tightens Trainium export controls and as the Linux Foundation's Agentic AI Foundation tries to keep MCP politically neutral — reshapes how Western clouds treat DeepSeek's open weights. V4 is expected in Q2. Whether it ships before or after Beijing has formal equity in the company is now the question that matters.

The only frontier lab still operating cheap. That stops today.

SiliconANGLE · Google Cloud Next 2026

Enterprise · Agentic

Google puts Gemini Enterprise at the center of an agentic taskforce.

At Cloud Next 2026 on Wednesday, Google reframed its agent story around a single product: Gemini Enterprise. The pitch is explicit about the pain: corporate AI agents today are siloed, hard to monitor, and lose context across long tasks. Google's answer is a shared agent plane that spans Workspace, Vertex, and now third-party connectors via MCP and A2A, with persistence designed to survive multi-day workflows — monthly reconciliations, multi-week sales cycles, SOX testing.

The architecturally interesting part is that Google is explicitly adopting MCP and A2A as the interop standard rather than pushing its own competing protocol. Six months ago that would have been a surprise. Today, with the Linux Foundation sitting at the center, it is a cost-of-entry. The real bet is that the observability, identity, and compliance layers around agents are where Google out-executes Anthropic and OpenAI on enterprise.

Persistence across days, not just turns, is the quiet feature everyone will copy.

Engadget · Bloomberg · Euronews

Incident · Governance

A Discord-linked group found its way into Claude Mythos.

Anthropic confirmed on Tuesday that it is investigating unauthorized access to Claude Mythos Preview — the same not-yet-GA model discussed two spreads up. According to Bloomberg, the intruders were not a nation-state team. They were a private online community who made an educated guess about the model's hosting URL based on Anthropic's naming conventions, then pivoted through a third-party contractor portal to reach it.

Nothing about the intrusion path is sophisticated. That is exactly why it matters. A model Anthropic has spent the last two weeks warning governments is capable of industrial-scale cyber offense was reached through URL guessing plus a vendor's single-factor login. The breach will make it harder for Anthropic to argue against external red-team mandates the next time Congress asks, and it is already reshaping the contractor-access checklist every frontier lab is quietly updating this week.

The model that finds zero-days was reached through a guessed URL.

That's a wrap.

Three of this week's ten stories flagged specifically for you — Opus 4.7, Managed Agents, and the MCP supply-chain advisory. The rest of the lineup tracks the infrastructure and capital shifts that set the context around them.

The throughline of the week is consolidation. Amazon's $25B into Anthropic and the ten-year Trainium commitment put Claude on the same long-duration footing that Azure gave OpenAI. Claude Managed Agents turns the Anthropic platform into a full hosting layer, not just a model. Opus 4.7 at xhigh by default inside Claude Code quietly raises the baseline for every coding agent shipping this year. On the other side of the Pacific, DeepSeek is for the first time taking outside capital, and Alibaba and Tencent are the likely first-movers.

The darker throughline is that agentic infrastructure is starting to surface the classical security problems that SaaS took fifteen years to learn. The MCP STDIO exposure, the Claude Mythos contractor-portal breach, and Anthropic's own warnings about frontier models in offensive cyber are three faces of the same realization: agents deployed at scale look a lot like an operating system, and nobody has written that operating system's hardening guide yet.
